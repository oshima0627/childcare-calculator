# 育児休業給付金シミュレーター

妊娠中・育児休業予定の方向けの手取り額シミュレーターです。月額給与と年齢を入力するだけで、育児休業給付金の詳細な計算結果を表示します。

## 🚀 機能

### 💰 正確な計算
- 厚生労働省の公式データに基づく計算ロジック
- 社会保険料・税金の詳細計算
- 標準報酬月額等級表による正確な保険料算出

### 📊 詳細な結果表示
- **現在の収支**: 手取り額と各種控除の内訳
- **育児休業給付金**: 1-6ヶ月目(67%)と7-12ヶ月目(50%)の詳細
- **比較チャート**: 通常時と給付金の視覚的比較
- **12ヶ月表**: 月別給付スケジュール
- **手取り維持率**: 現在の手取りに対する給付金の割合

### 🎯 ユーザー体験
- **リアルタイム計算**: 入力と同時に結果を表示
- **レスポンシブデザイン**: PC・スマホ完全対応
- **アクセシビリティ**: キーボード操作・スクリーンリーダー対応
- **FAQ**: よくある質問と詳細な回答
- **ソーシャル共有**: 計算結果をSNSで簡単共有

## 🛠️ 技術仕様

### フロントエンド
- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイル**: CSS Modules + CSS Custom Properties
- **状態管理**: React Hooks (useState, useEffect, useCallback)

### 計算ロジック
- **社会保険**: 健康保険、介護保険、厚生年金、雇用保険
- **税金**: 所得税（累進税率）、住民税（所得割・均等割）
- **給付金**: 賃金日額ベース、上限額適用

### SEO・パフォーマンス
- **メタデータ**: 構造化マークアップ、OGP、Twitter Card
- **フォント最適化**: preconnect、subset指定
- **画像最適化**: WebP/AVIF対応
- **CSS最適化**: 実験的最適化機能有効

## 📋 計算基準

### 使用データ（2025年度）
- **標準報酬月額等級表**: 厚生労働省公式データ
- **保険料率**: 健康保険10%、介護保険1.59%、厚生年金18.3%、雇用保険0.6%
- **給付金上限**: 賃金日額16,110円（2026年7月31日まで）

### 計算内容
```
賃金日額 = (月額総支給額 × 6) ÷ 180
1-6ヶ月目給付金 = 賃金日額 × 30 × 67%（上限323,811円）
7-12ヶ月目給付金 = 賃金日額 × 30 × 50%（上限241,650円）
```
